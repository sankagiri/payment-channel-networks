# payment-channel-networks
Code for the paper Pricing for Routing and Flow-Control in Payment Channel Networks
by Suryanarayana Sankagiri and Bruce Hajek
Published in IEEE/ACM Transactions on Networking

The following notebooks are used to generate the corresponding figures:
DEBT_control_mini_examples --> Figures 1 and 2 (need to change parameters in the notebook to get Fig 1)

DEBT_control_fivenode_network --> Figures 3 and 4 (need to change parameters in the notebook to get Fig 4)

DEBT_control_variable_demand --> Figure 5

DEBT_control_tennode_ERnetwork --> Figures 6 and 7
